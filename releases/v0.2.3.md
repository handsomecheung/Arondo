# Arondo v0.2.3

**Release Date:** 2026-07-07
**Commits:** 8
**Tag:** v0.2.3

Feature-rich update introducing task scheduling engine for future execution, automated queuing for follow-up user messages while the agent is busy, auto-rescheduling for quota-exhausted runs, interactive diff-view file folding, and client-side WebSocket heartbeat recovery.

---

## Highlights

- **Scheduled Tasks & Upcoming Queue**: Added support for scheduling project-scoped scripts to execute at a future fixed time. A dedicated "Upcoming" task section has been added to the Tasks dashboard, complete with a creation modal.
- **Auto-Queue Agent Follow-ups**: The chat input now remains active while an agent is running. Any follow-up user messages are automatically queued as `afterSession` scheduled tasks, which execute sequentially once the active process finishes.
- **Quota-Exhausted Auto-Rescheduling**: If an agent terminates due to quota limits, the system automatically schedules a `quotaAvailable` task to retry the last user prompt once the quota becomes available.
- **Diff View File Collapse/Expand**: Developers can now collapse and expand individual changed files dynamically inside the visual HTML diff viewer, simplifying the review of large changesets.
- **Runner WebSocket Connection Stability**: The Go runner client now performs heartbeat detection using WebSocket ping/pong and a read deadline, automatically initiating reconnection with exponential backoff if the link is severed.
