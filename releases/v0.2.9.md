# Arondo v0.2.9

**Release Date:** 2026-07-14
**Commits:** 5
**Tag:** v0.2.9

Global keyboard reload shortcuts, manual refresh buttons, WebSocket reconnection optimization, and automatic scrolling improvements. This version introduces Client-side manual reload options for PWA/standalone setups, eliminates redundant WebSocket reconnections on session switches, enhances automatic scrolling behaviors (including ResizeObserver bottom-pinning), and adds server version and runner connection times to the Settings and Runners dashboard.

---

## Highlights

- **Global Reload Shortcut & Refresh Buttons**: Added manual refresh buttons to the app header, Settings, and Tasks pages. A global keyboard shortcut (`F5` and `Ctrl/Cmd+R`) was registered for standalone/PWA users to reload the client.
- **WebSocket Reconnection Optimization**: Solved redundant client WebSocket reconnections when shifting sessions by storing the selected session ID in a ref and excluding it from the hook's dependency array.
- **Auto-scroll & Continuous Bottom Pinning**: Entering a session now instantly scrolls to the latest message (auto-scroll) instead of using smooth scrolling. In addition, `ResizeObserver` and `MutationObserver` monitor message cards for 2 seconds after entry to dynamically snap to the bottom as asynchronously rendered execution cards expand.
- **Runner Connection Time & Version Visibility**: Display runner connection start times (`connectedAt`) and the server version on the Settings and Runners pages, and relocated the Runner Access Control section to the bottom of the Settings page.
