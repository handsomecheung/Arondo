# Arondo v0.1.9

**Release Date:** 2026-07-03
**Commits:** 8
**Tag:** v0.1.9

Incremental release focused on mobile accessibility enhancements, navigation improvements, cleanup of unused features (GitHub PR integration), and UI refactoring. Key changes include a mobile-friendly terminal special-keys toolbar, navigation shortcuts for project-session switching, agent naming updates, and minor quota parsing fixes.

---

## Highlights

- **Mobile Terminal Special-Keys Toolbar**: Introduced a mobile-specific keyboard overlay ([TerminalKeyboardBar](file:///mnt/coder-workspaces/private-workspace/repos/github/Arondo/components/TerminalKeyboardBar.tsx)) inside the terminal modal, supporting crucial keys like ESC/TAB/CTRL/ALT, navigation arrows, and an FN layer (F1-F12). The layout dynamically docks above the soft keyboard using `visualViewport` tracking to prevent view obstruction.
- **GitHub PR Integration Removal**: Cleaned up the "Create PR" button and all related frontend and backend code to focus on core execution capabilities, reducing codebase complexity.
