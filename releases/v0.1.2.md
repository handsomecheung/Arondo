# Arondo v0.1.2

**Release Date:** 2026-06-26
**Commits:** 15
**Tag:** v0.1.2

Incremental release focused on task execution UX, session workflow improvements, terminal access, and runner visibility.

---

## Highlights

- **Restart/Retry actions**: Script tasks now have a Restart button; failed agent tasks have a Retry button — all replayed inline in the same terminal log without losing history.
- **User-stopped vs failed distinction**: Killing a task from the UI now produces a distinct "Stopped by user" status instead of a generic failure, with a clear separator in the terminal.
- **Slash commands in chat**: `/new` opens a new session with the same project/agent/runner; `/commit` sends a commit instruction to the current agent.
- **Open Terminal from session menu**: Every session's three-dot menu now includes an "Open Terminal" option for direct shell access on that session's runner.
- **Disconnected runners in settings**: The Nodes settings panel now shows disconnected runners with their last-seen time, so offline runners are never invisible.
