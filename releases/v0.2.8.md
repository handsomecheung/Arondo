# Arondo v0.2.8

**Release Date:** 2026-07-12
**Commits:** 9
**Tag:** v0.2.8

Session pinning, interactive sidebar menus, customized auto-archive settings, and visual brand updates. This version introduces session pinning to keep important tasks accessible, per-session dropdown menus, user-configurable session auto-archive day counts from the Settings UI, a new `/rename` slash command, and brand mark updates.

---

## Highlights

- **Session Pinning & Sidebar Menus**: Users can now pin important sessions to the top of the sidebar session list. Additionally, each session item in the sidebar and detailed view now features a three-dot menu, providing quick access to Pin, Rename, Archive, and Delete actions. The sidebar dropdown is rendered via a portal to prevent layout clipping.
- **Customizable Session Auto-Archive**: The number of idle days before active sessions are auto-archived can now be overridden and saved in the Settings page. This replaces the rigid environment-variable-only setup with a runtime-configurable input, stored in `~/.arondo/settings.json`.
- **`/rename <name>` Slash Command**: Added a new built-in slash command to easily rename the current active session directly from the chat input.
- **New Arondo Brand Mark**: Replaced the app favicon, icon, and header logo with the new Arondo brand mark, alongside updating page titles and subtitles to "Release from Anywhere, Any Device".
