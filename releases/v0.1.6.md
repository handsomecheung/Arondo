# Arondo v0.1.6

**Release Date:** 2026-07-01
**Commits:** 14
**Tag:** v0.1.6

Incremental release introducing intelligent auto agent selection based on quota remaining with cross-agent context handoff, manual agent switching within existing sessions, split implementation of execution cards (AgentExecCard and ScriptExecCard), enhanced prompt passing mechanism via environment variables/temporary files, and UI polishing.

---

## Highlights

- **Quota-based Auto Agent Selection & Context Handoff**: Implemented "Auto Model" selection that automatically scores and routes queries to the optimal agent based on current quota limits (e.g. Session/Weekly remaining for Claude, Gemini/Other limits for Antigravity). Context is automatically transferred across different agents during seamless handoffs.
- **Manual Agent Switching**: Added support for manual agent switching within active sessions, allowing users to pivot between agents dynamically.
- **Execution Cards Split (AgentExecCard & ScriptExecCard)**: Split `ExecCard` into specialized components. Agent execution cards now output inline PTY data (with CJK-aware width estimation) instead of needing a separate modal click.
- **Secure Prompt Passing**: Prompts are now passed to agents via temporary files and environment variables instead of exposing raw prompts in command-line arguments.
- **UI Enhancements**: Transitioned agent execution terminal displays from heavy xterm.js to lighter plain wrapped text views, integrated inline runner node detail panels under selected cards in the settings page, and added agent-specific icons (Claude, Antigravity, Codex, Shell).
