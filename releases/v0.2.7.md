# Arondo v0.2.7

**Release Date:** 2026-07-11
**Commits:** 9
**Tag:** v0.2.7

UI refinement, safety guardrails, and mobile UX improvements release. This version introduces project readiness warnings, a unified "+ TODO" session flow, mobile gesture support for deleting sessions, improved copy options for agent outputs, and several interface refinements.

---

## Highlights

- **Project Readiness Warnings**: Before creating a session or sending a message, Arondo checks if the project has uncommitted changes or a running agent. A warning modal prompts users with options to send anyway, auto-send once ready (via a codebaseReady scheduled task), or save the prompt as a TODO draft.
- **"+ TODO" Session Renaming**: Renamed "+ Draft" sessions to "+ TODO" sessions to better match task management semantics. The composer and sidebar updates distinguish between "TODO" (manual trigger) and "Pending" (scheduled auto-send) states.
- **Swipe-to-Delete on Mobile**: Users can now right-swipe a session item in the mobile sidebar to quickly delete it, sliding the session card to reveal a trash icon underneath.
- **Copy Actions for Agent Output**: Replaced the "Show Raw Output" code toggle in the `AgentExecCard` menu with explicit `Copy Output` (formatted markdown text) and `Copy Raw Output` (raw ANSI logs) options, improving utility for sharing and copying command results.
- **Selector UI Row Stacking**: Runner, Project, and Agent dropdown selectors in the New Session composer now stack vertically on separate rows, resolving overflow issues and creating a cleaner visual layout on narrow and mobile screens.
- **Auto-resize Chat Input**: The chat composer input area now automatically resizes to fit its placeholder text even when empty, ensuring readability and keeping the UI clean.
