# Arondo v0.2.10

**Release Date:** 2026-07-16
**Commits:** 11
**Tag:** v0.2.10

File uploads to session chat input, chunked file streaming in the Remote File Browser, unread completion indicators for sessions, and chat input draft persistence. This version enhances the smart chat input by allowing direct file attachment and uploading via WebSocket, removes the 512KB file size limit in the Remote File Browser by streaming file content in 64KB chunks on scroll, introduces unread visual indicators (colored dots and badges) for finished sessions, and ensures chat drafts are saved in localStorage per session.

---

## Highlights

- **File Upload to Session Chat Input**: Added an upload button to stage files as attachments. When sent, files are uploaded to the runner via WebSocket (`fs.upload`) and stored in a temporary directory, and their absolute paths are appended to the agent's prompt. The chat timeline displays a clean attachment note while hiding the local absolute path from the UI.
- **Chunked File Streaming in Remote File Browser**: Replaced the 512KB file browser preview limit with dynamic, scroll-based 64KB chunk streaming. File content is requested via offset and length query parameters, syntax-highlighted chunk-by-chunk, and appended seamlessly as the user scrolls.
- **Unread Session Completion Indicator**: The application now tracks when background tasks finish and compares `completedAt` with `lastViewedAt` timestamps. Unviewed finished sessions display a success (green) or error (red) dot in the sidebar, alongside a total count badge on the mobile navigation toggle.
- **Chat Input Draft Persistence**: Draft messages are now saved in `localStorage` keyed by session ID, preserving in-progress chat prompts when navigating between sessions or restarting the application.
