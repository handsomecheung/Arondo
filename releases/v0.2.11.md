# Arondo v0.2.11

**Release Date:** 2026-07-22
**Commits:** 15
**Tag:** v0.2.11

Sidebar project filtering, running scripts concurrently with active agents, project deletion protection for archived sessions, and mobile viewport resizing on Android Chrome. This version adds project filter chips below the Sessions/Projects toggle in the sidebar when sessions span multiple projects, enables running project scripts during agent execution, warns the user when deleting a project with archived sessions (while disabling unarchiving for sessions whose parent projects are deleted), and forces mobile browsers to resize content instead of scrolling when the software keyboard opens.

---

## Highlights

- **Sidebar Project Filtering**: Added horizontally scrollable project filter chips to the Sessions and Archived Sessions list in the sidebar. Visible only when sessions span two or more projects, selecting a chip filters the visible sessions to that project.
- **Run Scripts During Agent Execution**: Allowed project-scoped custom scripts to be started and executed concurrently while an AI agent is running, preserving the agent's running status during state transitions.
- **Project Deletion & Archiving Safety**: Warns the user in the confirmation dialog if a project has archived sessions that would become unusable after deletion, and disables the Unarchive button (showing a tooltip) for sessions whose parent projects have been deleted.
- **Live Indicator Dot in Task Queue**: Replaced the task queue badge count in the header with a breathing live indicator dot to signal active execution.
