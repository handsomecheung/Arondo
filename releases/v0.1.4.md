# Arondo v0.1.4

**Release Date:** 2026-06-28
**Commits:** 11
**Tag:** v0.1.4

Incremental release focused on config-driven custom agent commands, interactive Settings UI for commands, smart chat input UX (Tab completion, send on Enter), global project script execution, and Tasks dashboard layout & grouping improvements.

---

## Highlights

- **Custom & Config-Driven Slash Commands**: Agent slash commands are now config-driven. Users can define custom commands in `data/agent-commands.json` with regex matchers and templated message expansions (e.g. `/commit <message>`).
- **Agent Commands UI in Settings**: Added a complete management interface under Settings to list, add, edit, and delete custom agent slash commands from the web UI.
- **Smart Chat Input UX**: Improved chat input keyboard behavior (send message on `Enter`, newline on `Ctrl+Enter` / `Meta+Enter`) and added `Tab` completion to cycle through slash commands in the command menu.
- **Global Project Script Execution**: Added support for running project-scoped custom scripts globally (sessionless) directly from the project panel.
- **Tasks Dashboard Improvements**: The Tasks page now supports grouping tasks by **Scope** (Session or Project) or **Status** (Running, Completed, Stopped, Failed), and filtering by task type (Agent, Script, or Both). Task cards now display project/session subtitles.
- **Session Deletion Command**: Added a `/delete` slash command in chat to quickly delete the active session.
