# Arondo v0.2.4

**Release Date:** 2026-07-08
**Commits:** 4
**Tag:** v0.2.4

Security and usability update introducing Go runner WebSocket connection authentication, refactored token storage schema, optimized Tasks page layout, and refined mobile UX for executing custom scripts.

---

## Highlights

- **Runner Connection Authentication**: Added token-based validation for all incoming Go runner connections at `/runner`. The server checks the connection against a dedicated `runner` token in `tokens.json`. Runners can pass the token using the `--token` flag or the `ARONDO_RUNNER_TOKEN` environment variable.
- **Refactored `tokens.json` Storage**: Restructured `tokens.json` schema from a flat array to a clear structure separating client access tokens (`clients`) and the runner access token (`runner`). Old files are migrated automatically.
- **Tasks Page UI Layout**: Moved the `+Schedule` button next to the filter buttons (Scope/Status/Both) to improve logical placement and alignment on the Tasks management page.
- **Mobile Submenu Improvements**: Enhanced the mobile UX for the "Run Script" submenu with scrollable overflow handling, viewport containment, larger item touch-targets, and item dividers.
