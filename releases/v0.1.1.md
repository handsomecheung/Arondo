# Arondo v0.1.1

**Release Date:** 2026-06-25
**Commits:** 19
**Tag:** v0.1.1

Incremental release focused on agent session management, task execution improvements, and a new Tasks page with shell terminal support.

---

## Highlights

- **Agent session & resume**: Claude Code and Antigravity agents now support session persistence and resume, allowing interrupted coding tasks to continue where they left off.
- **Tasks page with shell terminal**: New dedicated Tasks page with an embedded shell terminal component connected via PTY WebSocket.
- **Task history & queue UX**: Task history retention across sessions, grouped by session with always-visible session names.
- **Concurrent script execution**: Same script can now run concurrently within a session.
