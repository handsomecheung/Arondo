# Arondo v0.1.5

**Release Date:** 2026-06-29
**Commits:** 9
**Tag:** v0.1.5

Incremental release focused on AI agent quota monitoring with tmux, global agent rules sync, chat input file browser picker trigger (@), file browser modal with syntax highlighting, task persistence refactoring, and navigation improvements.

---

## Highlights

- **AI Agent Quota Monitoring**: Added quota tracking for Claude and Antigravity agents. Quota statistics are collected periodically via tmux interactive sessions on runner nodes and displayed in Settings using progress bars.
- **Remote File Browser**: Integrated a file browser in the session menu featuring code syntax highlighting (via highlight.js), file preview support (up to 512KB), and a word wrap toggle option.
- **Global Agent Rules Sync**: Configured global agent rules via Settings UI, which are stored in `data/global-rules.md` and automatically synced to runner nodes (`~/.gemini/GEMINI.md` and `~/.claude/CLAUDE.md`) on runner registration.
- **Smart Chat Input File Picker (`@` trigger)**: Typing `@` in the chat input opens a file/directory picker modal, inserting the chosen path relatively into the chat input.
- **Task Metadata Persistence Refactor**: Replaced `data/active-tasks.json` with direct metadata serialization (pid, exitCode, runnerId, etc.) into `messages.json` of sessions and projects. Active tasks are recovered dynamically on server restarts.
